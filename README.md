
# Student Performance Analysis

## Description
This project analyzes student performance data to understand how different academic and behavioral factors affect students’ overall results.  
The main goal is to identify patterns between self-study hours, attendance, class participation, and final scores.

---

## Dataset
The dataset contains information about students and their academic performance.

Columns:
- `student_id` – Unique identifier for each student  
- `weekly_self_study_hours` – Hours spent studying per week  
- `attendance_percentage` – Attendance rate of the student  
- `class_participation` – Participation level in class  
- `total_score` – Overall academic score  
- `grade` – Final grade category
